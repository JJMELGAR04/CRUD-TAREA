# CRUD-TAREA
N-CAPAS


## Endpoints disponibles

| Método | URL | Descripción |
|--------|-----|-------------|
| POST | `/api/platos/create` | Crear un plato |
| GET | `/api/platos/getAll` | Obtener todos los platos |
| GET | `/api/platos/getBy/{id}` | Obtener plato por ID |
| PUT | `/api/platos/update/{id}` | Actualizar un plato |
| DELETE | `/api/platos/delete/{id}` | Eliminar un plato |
